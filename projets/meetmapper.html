MeetMapper: Connect with Friends All Over the World

Aller sur le site

Le projet

MeetMapper est un projet personnel que j'ai réalisé en 2019. Il s'agit d'un réseau social permettant de se connecter avec des amis partout dans le monde. L'objectif était de créer une application simple et intuitive, qui permette de voir où se trouvent ses amis et de discuter avec eux.

Technologies

  • Next.js: React framework.
  • Chakra UI: React component library.
  • React Hook Form: Form validation.
  • React: Javascript library for building user interfaces.
  • Next Auth: Authentication library for Next.js.
  • Openlayer: Map library.
  • Socket.IO: Real-time communication library.
  • Typescript: Javascript superset with types.
